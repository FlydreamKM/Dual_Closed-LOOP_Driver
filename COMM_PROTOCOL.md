# Dual_Closed-LOOP_Driver 通信协议（VOFA 调参版）

> **当前为 VOFA-only 模式**。所有通信均通过 **FireWater 文本命令** 下发，波形通过 **JustFloat** 协议回传。
> 待调试完成后，修改 `protocol.h` 中的 `PROTOCOL_VOFA_ONLY` 为 `0` 即可恢复完整二进制协议。
> UART2，115200-8-N1。

---

## 一、VOFA JustFloat 波形输出（下位机 → 上位机）

下位机上电后默认以 **200 Hz** 持续发送 JustFloat 帧，可直接在 VOFA 中绑定通道绘图。

### 数据通道定义（共 10 通道）

| 通道 | 名称 | 说明 |
|------|------|------|
| ch0 | M1_Speed | 电机1 实际速度 (rad/s) |
| ch1 | M1_Angle | 电机1 实际角度 (rad) |
| ch2 | M1_PWM | 电机1 PWM 输出 |
| ch3 | M1_TgtSpd | 电机1 轨迹目标速度 |
| ch4 | M1_TgtAng | 电机1 轨迹目标角度 |
| ch5 | M2_Speed | 电机2 实际速度 (rad/s) |
| ch6 | M2_Angle | 电机2 实际角度 (rad) |
| ch7 | M2_PWM | 电机2 PWM 输出 |
| ch8 | M2_TgtSpd | 电机2 轨迹目标速度 |
| ch9 | M2_TgtAng | 电机2 轨迹目标角度 |

### 修改输出频率

发送文本命令：
```
V 5     (200 Hz)
V 10    (100 Hz)
V 0     (关闭波形输出)
```

## 二、FireWater 文本调参命令（上位机 → 下位机）

所有命令均为 **ASCII 文本**，以换行符 `\n` 结束。可直接在 VOFA 的「命令」窗口或控件中使用。

### 命令速查表

| 命令格式 | 示例 | 说明 |
|----------|------|------|
| `M motor mode speed angle accel decel` | `M 0 0 10.0 0.0 20.0 20.0` | 设置目标参数 |
| `P motor pid_type kp ki kd` | `P 0 0 2.0 0.5 0.0` | 设置 PID |
| `C motor cmd` | `C 0 0` / `C 0 3` | 控制指令（使能/急停等） |
| `S motor` | `S 0` | 请求状态（仅在恢复二进制协议时有效） |
| `V interval_ms` | `V 5` / `V 0` | 设置 JustFloat 输出间隔 |

### 参数说明

- `motor`：`0`=电机1, `1`=电机2
- `mode`：`0`=速度模式, `1`=位置模式
- `pid_type`：`0`=速度环, `1`=位置环
- `cmd`：`0`=使能, `1`=失能, `2`=回零, `3`=急停, `4`=清除故障
- `interval_ms`：`0`=关闭, `5`=200Hz, `10`=100Hz

### 调参示例

```
C 0 0                    (使能电机1)
M 0 0 10.0 0.0 20.0 20.0 (电机1速度模式, 目标10 rad/s, 加减速20)
P 0 0 3.0 0.8 0.0        (电机1速度环PID: Kp=3 Ki=0.8 Kd=0)
```

### 双电机同步示例

```
C 255 0                  (使能双电机, motor_id=0xFF)
M 0 1 5.0 3.14 10.0 10.0 (电机1位置模式, 目标π rad, 限速5, 加减速10)
M 1 1 5.0 3.14 10.0 10.0 (电机2同样参数)
```

## 三、恢复二进制协议（调参完成后）

如需恢复原有的完整二进制帧协议（适合自定义上位机），只需修改一处：

**`Core/Inc/protocol.h`** 第 23 行：
```c
#define PROTOCOL_VOFA_ONLY      0   /* 将 1 改为 0 */
```

重新编译后，系统将恢复：
- 二进制帧收发（`0xAA 0x55` 帧头协议）
- 标准状态回传（`0x81` STATUS 帧）
- 心跳应答（`0x85` ACK 帧）
- 同时保留 VOFA JustFloat 和 FireWater 文本命令支持

## 四、编码器参数换算

- 编码器线数：1024
- 解码模式：TI12 四倍频
- 传动比：电机转 1 圈，编码器转 2 圈
- **电机 1 圈 = 8192 个 TIM 计数**

```
角度(rad) = encoder_total * 2π / 8192
速度(rad/s) = delta_count * 2π * 1000 / 8192   (控制周期 1 ms)
```

## 五、默认配置

| 参数 | 默认值 |
|------|--------|
| 控制周期 | 1 ms (1 kHz) |
| PWM 频率 | 20 kHz (TIM ARR = 1799) |
| VOFA JustFloat | 200 Hz (`vofa_interval_ms = 5`) |
| 标准二进制状态 | 关闭 (`status_interval_ms = 0`) |
| 速度环 PID | Kp=2.0, Ki=0.5, Kd=0.0 |
| 位置环 PID | Kp=5.0, Ki=0.0, Kd=0.1 |
| 默认加速度 | 10 rad/s² |
| 默认减速度 | 10 rad/s² |
